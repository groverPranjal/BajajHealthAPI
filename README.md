# Chitkara Qualifier REST API

Node.js Express API with two endpoints:

## Endpoints

- `GET /health` - Health check
- `POST /bfhl` - Operations: fibonacci, prime, lcm, hcf, AI

## Setup

```bash
npm install
node server.js
```

## Configure

Add your email and Gemini API key in `.env`:
```
OFFICIAL_EMAIL=your.email@chitkara.edu.in
GEMINI_API_KEY=your-api-key
```
